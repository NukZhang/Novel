# 故事主线设计与进展

## 核心主线：宇宙轮回与西行团队的使命

### 主线概述
故事围绕西行团队收集18个创世残片、打破由混沌海之主操控的宇宙轮回展开。西行团队表面是求取真经，实则是寻找并收集创世残片，最终帮助玉皇大帝完成最后一重劫难，打破宇宙轮回的枷锁，创造新的宇宙秩序。

### 宇宙轮回设定
- **宇宙周期**：每个宇宙包含诞生、演化、衰朽、毁灭四个阶段，由混沌海之主操控的生灭循环
- **大能者境界**：
  - 超脱境：超脱五行、跳出三界，如玉皇大帝、三清、盘古、镇元子等
  - 创世境：能创造小世界，如观音、如来等
  - 逍遥境：不受天条约束，如孙悟空等
  - 凡仙境：受天条约束的普通神仙
- **渡劫机制**：
  - 玉皇大帝需历经1750重劫难才能真正超脱宇宙轮回
  - 最后一重劫难需收集18个创世残片，打破混沌海之主的控制
  - 渡劫失败的大能者会被同化入混沌海，成为新宇宙的养分

### 创世残片设定
- **性质**：宇宙初开时遗留的核心神话力量载体，蕴含打破宇宙轮回的潜能
- **数量**：共18个（对应宇宙的18种基本元素）
- **分布**：散落于各界神话遗迹或势力盘踞地
- **效果**：觉醒上古神通，提供对抗混沌之力的力量
- **副作用**：消耗功德值、被上古意志侵蚀、触发天谴反噬

### 主线阶段划分

#### 第一阶段：初期（1-30章，50万字）
- **核心目标**：组建团队、熟悉西行号、完成前3个基础渡劫任务，初步接触创世残片与副作用
- **宇宙状态**：当前宇宙处于衰朽期，即将毁灭
- **残片进度**：0-5/18
- **核心事件**：
  - 玉皇大帝回顾过往劫难，决定寻找最后一重劫难的线索
  - 玄奘受观音点化，踏上取经之路
  - 孙悟空、猪八戒、沙僧、白龙马依次加入团队
  - 西行号启航，揭示其真正来历
  - 完成黑风山、高老庄、流沙河等基础任务
  - 初步接触创世残片及其副作用

#### 第二阶段：中期（31-65章，70万字）
- **核心目标**：深入核心遗迹区，遭遇多方势力围剿，团队内部出现分裂与背叛，关键成员牺牲，实力大幅提升
- **宇宙状态**：加速毁灭，大能者之间的竞争加剧
- **残片进度**：6-12/18
- **核心事件**：
  - 完成芭蕉扇、金光寺、小雷音寺等任务
  - 遭遇天庭暗流与混沌海之主的侵蚀
  - 团队内部出现分裂与背叛
  - 关键成员牺牲，团队重组
  - 实力大幅提升，获得更多创世残片
  - 了解到混沌海之主的存在与阴谋

#### 第三阶段：后期（66-90章，60万字）
- **核心目标**：揭开天庭暗流与创世残片的终极秘密，面临混沌海之主的直接威胁，团队重组并背水一战
- **宇宙状态**：即将毁灭，混沌海之主显露真身
- **残片进度**：13-18/18
- **核心事件**：
  - 发现西天真相，如来被囚禁
  - 获取玉皇大帝体内的光元素残片
  - 三清做出抉择，支持西行团队
  - 与混沌海之主展开终极决战
  - 孙悟空、猪八戒、沙僧、白龙马等核心成员牺牲
  - 玉皇大帝完成最后一重劫难，新宇宙诞生

#### 第四阶段：结局（91-100章，20万字）
- **核心目标**：完成使命，打破宇宙轮回，团队成员或牺牲或归隐，留下开放式或悲剧性收尾
- **宇宙状态**：旧宇宙毁灭，新宇宙诞生
- **残片进度**：18/18（全部收集完成）
- **核心事件**：
  - 新宇宙规则的确立
  - 观音留守引导新生命
  - 玄奘成为新宇宙的第一位守护者
  - 旧宇宙记忆的传承
  - 新守护者的培养与传承
  - 开放式结局，暗示新的开始
