# 各角色线设计与进展

## 核心角色详细设定

### L船长
- **全名**：林远征（极少人知道，船员只称"船长"或"L"）
- **年龄**：62岁（故事开始时，看起来只有45岁——碎片副作用之一）
- **背景**：前联邦海军上校，"猎户座"号巡洋舰舰长。32年前在一次秘密任务中发现工程级AI碎片，妻子（第一任大副）为保护碎片牺牲。他带着碎片叛逃联邦，创立Buddy号，表面做雇佣兵，实则寻找剩余碎片
- **外貌**：身材高大挺拔，两鬓微霜，眼神深邃如星空。左手无名指戴着一枚朴素的银戒——亡妻遗物
- **性格**：坚定果敢，深谋远虑，对船员如家人。极少发怒，但一旦发怒整艘船都会颤抖
- **特长**：舰队指挥、战略规划、人心洞察
- **口头禅**："我们是一家人"
- **异能**：[待觉醒] 时间感知——能预感未来3秒的危险，代价是每次使用都会加速衰老
- **隐藏设定**：他知道碎片完整后会发生什么，但从未告诉任何人。这是他最大的秘密
- **角色弧线**：
  - 卷1-2：神秘领袖，展示领导力与温情
  - 卷3-5：首个觉醒者，承受代价，开始明显衰老
  - 卷6-7：秘密逐渐暴露，船员信任危机
  - 卷8-10：真相大白，最终抉择——是否牺牲自己完成使命
- **与其他角色关系**：
  - 与大G：32年战友，唯一知道全部真相的人
  - 与戏命师：15年合作，科研层面的左膀右臂
  - 与老王：Buddy号最初的三人之一，生死之交
  - 与尼莫：发现其共鸣体质后格外关注，视为完成使命的关键人物
- **当前状态**：已登场(第6章)

### 大G（大副）
- **全名**：顾大伟
- **年龄**：54岁（看起来38岁——长期接触碎片的轻微影响）
- **背景**：前地球脱口秀演员，巅峰期全球前十。因在节目中揭露某财阀黑幕，被全面封杀并遭追杀。L船长救了他，从此追随至今。他是船上除船长外唯一知道碎片全部秘密的人
- **外貌**：戴金丝眼镜，头发梳得一丝不苟，永远穿着剪裁得体的深色制服。笑起来很有感染力，但眼底总有一丝看透世事的疲惫
- **性格**：细致谨慎，幽默感强，是船上的定海神针。看似随和，实则心思缜密，能在谈笑间化解危机
- **特长**：后勤管理、人际调解、信息分析、危机处理
- **口头禅**："这个嘛，让我想想怎么说……"（拖口秀演员的习惯）
- **隐藏设定**：他揭露的那个财阀，与联邦高层有千丝万缕的联系，这条线会在中后期引爆
- **角色弧线**：
  - 卷1-3：团队守门员，展示管理才能与幽默
  - 卷4-6：过去被揭开，财阀势力找上门，面临抉择
  - 卷7-10：在船长衰弱后逐渐承担更多领导责任
- **与其他角色关系**：
  - 与L船长：绝对信任，共同守护秘密32年
  - 与阿远：内外分工的管理搭档，亦师亦友
  - 与尼莫：第一个正式接纳尼莫的人，暗中观察其成长
- **当前状态**：已登场(第1章)

### 戏命师（首席科学家）
- **全名**：季鸿渊
- **年龄**：42岁
- **背景**：前银河科学院最年轻的终身院士，研究领域横跨量子物理、神经科学、人工智能。15年前因"违规实验"被除名——实际上是他发现了工程级AI的存在，科学院高层为掩盖真相将他驱逐。L船长找到他，用碎片研究机会换取他的加入
- **外貌**：英俊儒雅，永远穿着白大褂，手指修长如钢琴家。眼镜后的眼睛总是带着探究的光芒
- **性格**：平稳无争，博学好为人师，对科学有近乎偏执的热爱。看似温和，实则极度固执——一旦认定的事绝不改变
- **特长**：全领域科学研究、遗物分析、神经同步技术
- **口头禅**："从科学角度来说……"
- **隐藏设定**：他研究神经同步的真正目的，是想创造人机融合体——让人类意识与AI融合。这个想法源于他死去的妹妹（脑死亡但身体存活）
- **角色弧线**：
  - 卷1-4：智力担当，解密遗物，推进神经同步研究
  - 卷5-7：与遗物产生特殊联系，开始听到"声音"
  - 卷8-10：面临终极选择——科学理想vs人性底线
- **与其他角色关系**：
  - 与L船长：合作15年，互相需要但并非完全信任
  - 与白小姐：[待发展，可能的感情线]
  - 与尼莫：发现尼莫的共鸣体质，视为重要研究对象/合作者
- **当前状态**：已登场(第9章)

### X先生（机甲战士队长）
- **全名**：夏铭（军中代号"破晓"）
- **年龄**：40岁
- **背景**：前联邦陆战队王牌机甲战士，服役12年，击坠记录217台，军衔上校。在一次任务中被上级命令放弃受困战友，他违抗命令救人成功，却被军事法庭判处叛国罪。L船长在他被押送途中将他救出，从此效命Buddy号
- **外貌**：身材魁梧，面容刚毅，左脸有一道从眉角延伸到下巴的旧伤疤（救战友时留下）。平时表情严肃，但笑起来会露出虎牙
- **性格**：铁血硬汉，纪律严明，对战士要求极高。但内心柔软，会偷偷给受伤的队员送补品
- **特长**：机甲战斗全能（尤擅重型）、战术指挥、近身格斗
- **口头禅**："战场上，纪律就是生命"
- **机甲**：破晓（重型突击机甲，全船火力最强）
- **隐藏设定**：他当年救出的战友中，有一个后来成为了联邦机甲部队的将军——这个人一直在暗中保护他，也在寻找他
- **角色弧线**：
  - 卷1-3：战斗核心，展示战术素养，开始训练尼莫
  - 卷4-6：旧日战友找上门，联邦势力介入，面临立场考验
  - 卷7-10：[重大事件——可能牺牲或叛变的关键节点]
- **与其他角色关系**：
  - 与飞哥：最信任的副手，战场上的左膀右臂
  - 与铁锤/闪电：严格的上司，但私下关心
  - 与尼莫：起初不看好，后逐渐认可其潜力
- **当前状态**：已登场(第6章)

### 尼莫（视角主角）
- **全名**：尼莫（真名不详，可能是孤儿）
- **年龄**：26岁（故事开始时）
- **背景**：蓝星理工计算机系毕业，做了三年机甲格斗游戏开发，公司倒闭后被裁员。躲赌债时误上Buddy号。看似普通人，实则拥有极其罕见的"共鸣体质"——能与工程级AI碎片产生共鸣
- **外貌**：中等身材，相貌普通，眼神里总带着一点游离。穿着随意，头发经常乱糟糟的
- **性格**：散漫随性，偶有小聪明，适应力极强。看似没心没肺，实则观察力敏锐。不喜欢麻烦，但关键时刻绝不退缩
- **特长**：游戏开发思维（转化为机甲设计直觉）、快速学习、共鸣体质
- **口头禅**："我就随便看看……"
- **隐藏设定**：他的共鸣体质并非天生，而是与他失踪的父母有关——他父母可能是工程级AI项目的研究员
- **成长路线**：机甲设计部实习 → 戏命师助手 → 机甲战士（后期）
- **角色弧线**：
  - 卷1-2：菜鸟期，适应船上生活，展现设计天赋
  - 卷3-5：成长期，共鸣体质觉醒，开始接触碎片秘密
  - 卷6-8：成熟期，成为机甲战士，独当一面
  - 卷9-10：见证者，记录Buddy号的最终命运
- **与其他角色关系**：
  - 与飞哥：亦师亦友，最投缘的人
  - 与大J：从被忽视到被认可
  - 与小J：同期新人，从竞争到并肩
  - 与L船长：被特别关注，承载着某种期望
- **当前状态**：已登场(第1章)

- **全名**：方飞（军中代号"幽灵"）
- **年龄**：35岁
- **背景**：前星际特种部队成员，代号"幽灵"，专精单人渗透任务，成功率100%。在一次任务中全队覆灭，只有他活了下来，但被怀疑叛变。他没有辩解，选择消失，三年后被L船长找到
- **外貌**：身材不高但肌肉线条分明，面容普通容易被忽略（特种兵的优势）。眼神锐利如鹰，但笑起来很有亲和力
- **性格**：豪爽直接，重情重义，战场上冷酷无情，战场下温暖如兄长。有轻微的幸存者愧疚
- **特长**：渗透作战、情报收集、各类机甲驾驶、近身格斗
- **口头禅**："活着就是最大的胜利"
- **机甲**：幽灵（中型全能机甲，擅长隐蔽突袭）
- **隐藏设定**：他的队伍覆灭并非意外，而是被人出卖。出卖者至今逍遥法外，这是他心中最大的刺
- **角色弧线**：
  - 卷1-4：尼莫的引路人，展示老兵的智慧与温情
  - 卷5-7：发现当年真相，追查叛徒
  - 卷8-10：[关键抉择——复仇vs放下]
- **与其他角色关系**：
  - 与X先生：最信任的上司，战场搭档
  - 与尼莫：亦师亦友，看到了年轻时的自己
  - 与铁锤：老战友，互相信任
- **当前状态**：已登场(第5章)

- **年龄**：44岁
- **背景**：银河系机甲设计排名前五十的天才，曾是联邦最高机密"神盾计划"的首席设计师。在发现计划真正目的（制造能毁灭星球的超级机甲）后，他销毁了所有资料并叛逃。联邦悬赏他的人头已经十五年
- **外貌**：头发乱糟糟，眼睛下永远挂着黑眼圈，工作服皱巴巴沾满油渍。但双手极其稳定，像外科医生
- **性格**：孤僻内向，专注工作到近乎偏执，不善交际。话少但每句都在点子上。看似冷漠，实则极度负责——他设计的每一台机甲都当作自己的孩子
- **特长**：机甲设计与改造（全银河顶尖）、结构力学、材料科学
- **口头禅**："别太有感情"（实际上他自己最有感情）
- **隐藏设定**：他住在设计部、从不休息，是因为他在秘密设计一台机甲——用来对抗"神盾计划"可能造出的怪物
- **角色弧线**：

  - 卷5-7："神盾计划"重启，联邦势力追杀加剧
  - 卷8-10：他的秘密机甲完成，关键战役
- **与其他角色关系**：
  - 与小J：表面冷淡，实则在暗中培养
  - 与尼莫：从忽视到认可，最终传承
  - 与L船长：互相尊重，船长是少数理解他的人
- **当前状态**：已登场(第2章)
